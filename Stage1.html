World Wide Web and HTML

The Web/Internet is the world's largest computer network.

Major pieces of the Web: user/computer+browser --- Internet --- servers

A Web Page is a text document written in a language called HTML. Web browsers read these HTML documents, and then interpret and display them.

HTML

HTML - HyperText Markup Language is a system for tagging text files to achieve font, color, graphic, and hyperlink effects on Web pages. HTML is the main document type on the Web.

HTML is the language of the Web, and it controls the "structure" of a Web page.

Inline vs Block Elements

In-line elements take up only as much width as needed, and do not force new lines. Block elements form an "invisible box" around the content inside of them.

Examples of in-line elements are span and br.

Examples of block elements are p and div.


Lesson 2: Creating a Structured Document with HTML

DOM - The "tree" structure of HTML

The browser turns tags into elements that form the tree, because of the Document Object Model (DOM).

When programmers talk about the "DOM" (Document Object Model) they are talking about the tree-like structure of a page.

The Document Object Model (DOM) is a convention for representing and interacting with objects in HTML. The nodes of HTML are organized in a tree structure, called the DOM tree.

HTML boxes

Every element on a page is a rectangular box and may have width, height, padding, borders, and margins. Boxes are create with div tags.

The box model in CSS

Content is the image or text.
Padding clears an area around the content.
Border goes around the padding and content.
Margin clears an area around the boarder, padding, and content.

Dealing with sizing issues in CSS

There are two techniques that can be used to help deal with sizing issues:
1. Set sizes in terms of percentages rather than pixels
2. Set the box-sizing attribute to border-box for every element in CSS
Example in CSS:

*{
box-sizing: border-box;
}

Why Computers are Stupid

Computers are stupid because they execute commands literally, so a programmer cannot be ambiguous when coding.

The Importance of Avoiding Repetition

A programmer could apply style parameters to each element in HTML, which could lead to repetition of code. By using Cascading Style Sheets (CSS) a programmer can apply style parameters to classes, then associate elements to classes. This eliminates the repetition of style coding, thus allowing to the pages to load quicker.
